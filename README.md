
# Método 90D - Landing Page (Vite + React)

Arquivos gerados automaticamente para sua landing page do Método 90D.
- Para rodar localmente:
  1. Instale dependências: `npm install`
  2. Rode em dev: `npm run dev`
- O botão de compra redireciona para o WhatsApp com a mensagem pré-definida.
